Sobre o Projeto

Este site é um aprimoramento de um projeto já existente, mantendo sua base e estrutura principal. O objetivo foi transformar um site simples em algo muito mais moderno, organizado e visualmente bonito, utilizando HTML e CSS para melhorar completamente sua aparência.

Por incrível que pareça, este site ainda utiliza a mesma base do projeto original, porém com diversas melhorias visuais, organização de código e estilização moderna.
